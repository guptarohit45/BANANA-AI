☕ Café Management System (Python Tkinter)

A GUI-based Café Management System built using Python and Tkinter.
This application allows users to select menu items, calculate the bill, generate receipts, and manage orders efficiently.
This project is useful for beginners learning Python GUI development and basic billing system logic.

📌 Features

-☕ Menu-based ordering system
-➕ Enter quantity for each item
-💰 Automatic bill calculation
-🧾 Receipt generation
-🧮 GST (5%) tax calculation
-🔄 Reset order option
-❌ Exit confirmation dialog
-🖥️ Simple and user-friendly GUI

🧰 Technologies Used

-Python
-Tkinter (Python GUI Library)
-Random Module (for bill number)
-Time Module (for date)

📊 How It Works

1)User enters the quantity of items from the menu.
2)Click Calculate Total to compute subtotal and tax.
3)Click Generate Receipt to display the detailed bill.
4)The receipt includes:

a)Bill Number
b)Date
c)Ordered items
d)Subtotal
e)Tax
f)Final Total

5)Use Reset to clear the order.
6)Use Exit to close the application.

🖥️ Application Interface

The GUI contains three main sections:

1️⃣ Menu Section

1)Displays café items and prices
2)Allows user to enter item quantities

2️⃣ Bill Summary

1)Shows receipt
2)Displays subtotal, tax, and total

3️⃣ Control Buttons

1)Calculate Total
2)Generate Receipt
3)Reset Order
4)Exit Application
